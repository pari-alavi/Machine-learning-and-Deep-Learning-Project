# Machine Learning and Deep Learning Project

## Egocentric Vision

Egocentric videos are typically long and unstructured, making information retrieval challenging.
This project enhances temporal localization by generating textual answers from relevant video segments, enabling efficient query processing and improving interpretability in video understanding.

---

## 🚀 Features

* **Two Model Architectures**

  * **VSLBase** (simplified baseline)
  * **VSLNet** (with Query-Guided Highlighting)

* **Multimodal Support**

  * Compatible with **Omnivore** and **EgoVLP** features

* **End-to-End Pipeline**

  * Seamless processing from input video to final textual output

* **Core Capabilities**

  1. Localizes relevant video segments
  2. Generates textual answers
  3. Evaluates performance using ROUGE and METEOR metrics

* **Optimized for Egocentric Video**

  * Handles long, unstructured first-person recordings
  * Focuses computational resources on key moments

---

## 📊 Evaluation

The model performance is evaluated using:

* **ROUGE** (Recall-Oriented Understudy for Gisting Evaluation)
* **METEOR** (Metric for Evaluation of Translation with Explicit ORdering)

---

## 🧠 Key Highlights

* Designed specifically for **egocentric (first-person) video understanding**
* Improves **interpretability** through natural language outputs
* Efficient handling of **long-duration video content**
